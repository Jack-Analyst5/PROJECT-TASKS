Project Overview
This project is part of my journey in learning Python.

Goal
To determine which factors contribute the most to a movie's success.

Description
This project demonstrates my ability to apply Python for data analysis. I use:

Python for core programming
Pandas for data manipulation
Matplotlib and Seaborn for data visualization
The dataset is stored in Excel and imported into a Microsoft Fabric Notebook. It represents real-world retail transaction data, including sales, invoices, and product information across multiple stores.

Skills Applied
Data cleaning
Exploratory Data Analysis (EDA)
Correlation analysis
Hypothesis testing
Data visualization

Technologies
Python
Pandas
Matplotlib
Seaborn
Microsoft Fabric
Results

Using Python and EDA techniques, I explored relationships between variables and identified key factors that influence outcomes. Visualizations helped highlight trends, correlations, and patterns within the dataset.
